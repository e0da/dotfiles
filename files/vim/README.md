dotvim
======

This is my vim configuration. I use [Vundle][] to manage plugins.

Installation
------------

```sh
git clone https://github.com/justinforce/dotvim ~/.vim && ~/.vim/install_vundle && ln -s ~/.vim/vimrc ~/.vimrc
```
